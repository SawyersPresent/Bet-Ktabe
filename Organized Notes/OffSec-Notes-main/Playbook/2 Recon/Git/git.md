# git

Software development tracking

## Capabilities

```bash
# Initialize a repo
git init <dir>

# Check repo status
git status

# Display branches
git branch

# Change branches
git switch <branch>

# Show commits within the current branch
git log

# Show changes related to a specified commit
git show <commit_id>

# Restore to previous commit
git checkout -- .
```

# TLDR;


Sudo privs can be granted to an account allowing them to run certains commands as root or any other account, this doesnt change any of the users original privileges or grant excessive ones. The system checks `/etc/sudoers/` to see if they can execute said command, 




```powershell
// ------------------------------------------ Sudo Rights Abuse ------------------------------------------ //
sudo -l

GTFOBins

read lol





```
